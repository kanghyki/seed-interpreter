# Seed Interpreter

A programming language interpreter implementation based on the Lox language from [Crafting Interpreters](https://craftinginterpreters.com).

## References

- **Book**: [Crafting Interpreters](https://craftinginterpreters.com) (free online)
- **Original Repository**: [github.com/munificent/craftinginterpreters](https://github.com/munificent/craftinginterpreters)
- **Author**: Robert Nystrom

## Requirements

- Java 22 or higher

## Building and Running
```bash
# Run REPL (interactive mode)
make run

# Run with a script file
make run seed/example.seed
```
